#Sproutify backend

## Frontend side of project created for 2021 Mini Hackathon Nowa Akademia

To run app install node js 1.16.10, npm and type following commands:

##### git clone https://github.com/arosicki/hackathon-frontend.git

##### npm install

##### npm start
